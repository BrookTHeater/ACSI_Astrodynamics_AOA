To open and run the Natural Reentry Analysis:

Must have: GMAT INSTALLED TO DEVICE

1. Open the Automated GMAT Rentry Analysis folder
2. Read API_README.txt 
3. Open load_gmat.py
4. Update load_gmat.py with File Path to GMAT bin folder
5. Open ReentryCalculation.py
6. Run
